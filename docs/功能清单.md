## 3. MVP 范围

24 小时内建议优先完成以下范围：

- 后端支持 PDF 上传、解析、清洗
- 使用 AI 提取结构化信息
- 支持输入 JD 并输出匹配评分
- 前端完成上传、输入、展示结果三个核心流程
- README 写清楚架构、接口、部署和演示方式
- 线上前端可访问，后端接口可调用
